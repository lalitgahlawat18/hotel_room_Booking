# hotel_room_Booking
Introduction

Hotel Management System is a C++ based application developed to manage hotel room booking records. The project helped me understand Object-Oriented Programming concepts, file handling, and CRUD operations.

Working

The system allows hotel staff to create customer booking records, view existing records, update customer information, and delete records when required. All records are stored using file handling so that data is preserved even after the program closes.
